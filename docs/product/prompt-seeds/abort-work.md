# `run:abort` (reserved, no-agent path)

Для `run:abort` агентный job не запускается и prompt не рендерится.

Лейбл `run:abort` должен обрабатываться платформой напрямую:
1. остановить активный run;
2. удалить runtime namespace (если существует);
3. записать audit/flow-события и обновить статус выполнения.
