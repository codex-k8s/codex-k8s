# Якоря и чанки книги

Карта заголовков `docs/source/book.md` для поэтапного извлечения содержания.

## Как работать с книгой частями
- Макро-чанк: глава (`###`) или часть (`### ЧАСТЬ ...`).
- Мезо-чанк: подразделы (`####`) внутри выбранной главы.
- Микро-чанк: подпункты (`#####`) для точечных цитат и аргументации.
- Для ссылок используйте формат GitHub line anchor: `.../book.md#Lx-Ly`.
