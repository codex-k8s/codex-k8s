# Пакет рекомендаций 1:1

Этот каталог содержит рекомендации по доработке документов на основе `docs/source/book.md`.

## Что внутри
- `book_best_practices.md` — свод лучших практик и опорные ссылки на книгу.
- `design-guidelines/**.recommendations.md` — рекомендации для документов `docs/design-guidelines/*` (1:1).
- `templates/**.recommendations.md` — рекомендации для шаблонов `docs/templates/*` (1:1).

## Покрытие
- Целевых документов: `75`.
- Сгенерированных рекомендаций 1:1: `75`.
